Data Science Libraries

Seaborn

Seaborn is a Python visualization library based on matplotlib that provides a high-level interface for drawing attractive statistical graphics.

Statsmodels

Statsmodels is a Python module that allows users to explore data, estimate statistical models, and perform statistical tests.

NumPy

NumPy is the fundamental package for scientific computing with Python, providing support for arrays, mathematical functions, and more.
